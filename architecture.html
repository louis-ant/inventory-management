Factory Inventory Management System

System architecture, tech stack, and request flow

Tech Stack

Frontend

  • Vue ^3.4.21
  • Vue Router ^4.3.0
  • Axios ^1.6.7
  • Vite dev server

Backend

  • FastAPI ≥0.110.0
  • Pydantic ≥2.5.0
  • Uvicorn ≥0.24.0
  • Python uv managed

Data & Testing

  • JSON files in-memory
  • pytest ≥8.0.0
  • httpx TestClient
  • No database mock data
